# 🌤️ SkyCast Pro - Weather App

Una aplicación de clima moderna, rápida y con un diseño elegante (Glassmorphism) para consultar el tiempo en cualquier ciudad del mundo en tiempo real.

## 🚀 Características
* **Datos en tiempo real:** Obtiene información actualizada de la API de OpenWeather.
* **Diseño Moderno:** Interfaz responsiva utilizando **Tailwind CSS**.
* **Iconos Dinámicos:** Cambian según el estado del clima (sol, lluvia, nieve, etc.).
* **Detalles precisos:** Muestra temperatura, sensación térmica, humedad y velocidad del viento.

## 🛠️ Tecnologías utilizadas
* HTML5
* Tailwind CSS
* JavaScript (ES6+)
* Font Awesome (Iconos)
* OpenWeather API
